<Bq>드리프트를 얕게 한 상태에서 드리프트가 끊어지지 않도록 방향키로 계속 유지시켜주는 기술.</Bq>

### 개요

오리지널 카트라이더에서 발견된 기술로, 원래는 버그였으나 선수들도 해당 기술을 리그에서도 사용하게 되면서 공식 기술로 인정받게 되었습니다.

명칭은 드리프트를 길게 끈다하여 끌기 혹은 드리프트를 아슬아슬하게 끊기지 않도록 방향키로 톡톡 쳐준다하여 톡톡이 라고도 불리었습니다.

### 발생 원인

가장 큰 원인은 카트바디의 **드리프트 탈출력** 입니다.

드리프트 탈출력이란, 이름 그대로 드리프트에서 빨리 탈출하여 카트바디가 끌리지 않게 하는 것을 의미합니다.

<br/>

카트라이더 물리엔진 특성 상 드리프트 탈출력이 라는 것이 카트의 드리프트를 끊어지게 하려면, 

카트의 속도를 증가시켜 전진성을 높여 드리프트를 할 수 있는 각도에서 벗어나게 해 드리프트를 끊기게 하는 것입니다. 

<br/>

이러한 특성 때문에 드리프트 탈출력이 높은 카트바디를 타고 드리프트가 끊기지 않도록 아슬아슬하게 유지만 해주면

**드리프트를 끊으려고 카트의 속도를 증가시키려는 힘**이 기존의 속도에 더해져 직진 부스터를 뛰어넘는 속도를 만들어내는 것입니다.

<br/>

따라서, 드리프트 탈출력이 높을수록 톡톡이의 가속도와 최고속도가 높아지게 됩니다.

### 조작법

오리지널 카트라이더와 방법은 비슷합니다.

최적화 드리프트 후, 드리프트가 끊어지지 않도록 방향키를 살짝만 누르면서 유지시켜주면 됩니다.

<br/>

방향키를 쎄게 누르게 되면 오히려 감속을 먹게되니 살짝만 눌러주고,

드리프트 키는 드리프트 이후 누르지 않습니다.
