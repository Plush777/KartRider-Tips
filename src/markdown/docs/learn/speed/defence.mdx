<Bq>주행 중 상대의 주행을 지속적으로 방해하는 기술.</Bq>

다른 명칭으로 블로킹이라고도 하며, 일부러 속도를 줄이거나 멈춰서 상대를 막는 경우에는 스탑이라고 부릅니다.

<br/>

개인전에서는 1등을 확실히 하기위한 수단에서 사용되며, 

팀전에서는 아군과의 거리 유지 혹은 승리를 확정하는데 주로 사용됩니다.

<br/>

상대방의 움직임을 파악할 줄 알아야하고, 실패 시 하위권에 빠질 수 있으며,

성공 하더라도 상대방이랑 충돌하는 것이기 때문에 잘못하면 디펜스 하려다 

상대방이랑 같이 사고에 휘말리는 경우도 있으니 신중히 해야합니다.
