<Bq>랍토르 L이 이름에 걸맞게 스티어를 개조하여<br/>
왼쪽으로 방향을 조작할 때마다 스킬 게이지가 차오릅니다.<br/>
속도가 빠를수록 게이지가 더 빠르게 차오르며<br/>
게이지가 가득 차면 바리케이드를 획득합니다.</Bq>

좌회전이 아닌 왼쪽 방향키 입력이 기준임에 주의 해야합니다. 
즉, 우회전 드리프트 중 왼쪽 방향키를 누르면 게이지가 충전되는 형식입니다. 

<br/>

바리케이드 특성 상 트랙 구조에 영향을 크게 받고, 누른 시간 만큼만 충전되므로 실제 쿨타임은 표기보다 더 긴 편이라 <br/>
재미용 외엔 타 캐릭터들에 비해 실용성이 거의 없어서 사실상 없는 캐릭터로 취급됩니다.
