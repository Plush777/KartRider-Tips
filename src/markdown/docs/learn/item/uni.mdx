<Bq>우니가 자신의 레이싱 실력을 증명할 기술을 만들어 선보입니다.
특정 레이서를 선택해 달려가며 만들어낸 회오리는 <br/>
충돌한 상대 레이서를 강하게 튕겨냅니다.<br/><br/>
스킬 사용 중에 상대 레이서 1명당 1회만 공격할 수 있습니다.<br/>
선택한 유저를 스킬로 공격 성공하거나 지속 시간이 끝날 경우 우니 스킬이 종료됩니다.<br/>
스킬 사용 종료 시 일시적으로 감속합니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539">2024/5/30 패치</Link> 이후로 스킬이 추가되었습니다.</Bq>


사용 시 조준한 상대를 향해 자신이 정신없이 회전하면서 큰 가속을 받습니다. 가속 메커니즘은 자석과 비슷합니다. <br/>
빠르게 상대방한테 붙은 다음 멀리 날릴 수 있어서 미들 순위 디펜스용으로도 좋으나 쿨타임이 꽤 길고, 어떻게 충돌하는지에 따라 타격을 별로 못주거나 앞으로 튕겨나갈 때도 있어 <br/>
리턴이 큰 만큼 리스크도 큰 편입니다. 

<br/>

스킬 종료 후 감속의 정도는 번개 아이템을 피격당했을 때 수준이지만, 가속 과정에서 다수의 유저에게 공격이 성공적으로 끝나고 난 이후라면 크게 체감은 안되는 수준입니다. <br/>
공격 아이템을 맞으면 스킬이 바로 끝나기 때문에 넓은 범위를 견제할 수 있는 비비, 토비, 아이리 등의 캐릭터가 우니의 카운터로 볼 수 있습니다.

또한, 조준한 상대가 어떠한 이유로 멈추거나 뒤쳐지면 가속을 제대로 받기 전에 스킬이 종료되거나 상황에 따라선 스킬이 끝나지도 않았는데 속도가 거의 0인 수준으로 <br/>
느려지는 치명적인 약점이 존재하므로 사기성도 있지만 그만큼 리스크가 큽니다.

<br/>

네오와 동일하게 우니의 튕겨내는 효과는 팀원에게 적용이 안되지만, 팀원을 조준하여 가속을 받는 것은 가능합니다.
