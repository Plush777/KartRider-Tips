<Bq>자신과 일정 거리 앞에 물폭탄을 던집니다.<br/>
반경 내에 있는 레이서는 물에 갇혀 잠시 조작이 불가능해집니다.</Bq>

물폭탄에 갇혔을 경우 좌우 키를 연타하면 빠르게 탈출할 수 있으며, <br/>
물폭탄은 항상 중앙에 떨어지기 때문에 넓은 맵에서는 사이드로 가면 피할 수 있습니다.

<br/>

팀전에서 물폭탄의 색깔이 사용한 사람의 팀 색깔로 나타나며, 공격은 아군 적군 둘 다 가능하기 때문에 <br/>
팀원에게 피해를 주지않도록 주의해야 합니다.<br/>
보이스 채팅을 하고 있을경우 팀원에게 사이드 타라, 혹은 실드를 써라 라고 말해주는게 좋습니다.

### 관련 용어

- 물 : 물폭탄의 줄임말.
- 물거리 : 물폭탄을 던지는 거리. 또는 1등과의 거리를 표현할 때 사용되기도 함. <br/>
ex) (상대가 1등일 때) : 1등 물거리 돼요 / (우리 팀이 1등일 때) : 물거리 안돼요. 쭉 달려요~
- 꽉물 : 좁은 길의 경우 물폭탄을 피할 수가 없어서 방어 아이템이 없으면 무조건 맞을 수 밖에 없는데, 이때 사용하는 용어.
- 칼물 : 물거리를 정확하게 계산하여 상대 머리 위에 바로 떨어지도록 물폭탄을 던지는 기술.
