<Bq>아이리가 물폭탄을 한번에 여러 개 투척할 수 있는 <br/>
다발 물폭탄으로 개조하여 레이싱에 참여합니다.<br/>
다발 물폭탄을 지닌 상태로 게임을 시작하며<br/>
물폭탄 획득 시 다발 물폭탄으로 변경됩니다.<br/>
다발 물폭탄 사용 시 물폭탄의 개수는 일정 확률로 등장합니다. <br/>
아이리는 일정 시간마다 물폭탄을 획득합니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539">2024/3/21 노르테유 패치</Link> 이후로 게임 시작 시 다발 물폭탄을 소지하도록 변경되었습니다.</Bq>

물폭탄 획득 시 다발 물폭탄 등장 자체는 확정이지만, 다발 개수에 낮은 확률이 적용되어 있어 처음 지급되는 다발 물폭탄을 제외하고 <br/>
5개 다발 물폭탄은 게임 내내 획득하기가 어렵습니다. <br/>

다발 물폭탄의 범위는 일반 물폭탄보다 약간 작은 수준입니다. 추가로, 다발 물폭탄은 **물거리를 알려주지 않기 때문에** 직접 물거리를 재서 던져야 합니다. <br/>

이러한 부분 때문에 실제 난이도는 중급에서 상급 정도로 평가받고 있습니다.

<br/>

팀전에서는 잘못 던지면 팀킬의 위험이 크기 때문에 주의해야 합니다.

<br/>

단점으로는 물거리조차 안나올 정도로 멀리 뒤쳐지면 답이 없고, 템을 가지고 있는 상태에서 각을 재고 써야하기 때문에 <br/> 
아이템을 바나나로 바꾸는 크리스에게 취약합니다.
