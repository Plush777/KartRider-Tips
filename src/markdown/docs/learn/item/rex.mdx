<Bq>렉스가 자신을 공격한 대상을 배신자로 간주하여 끈질긴 복수를 시행합니다. <br/>
공중에 뜨는 공격에 피격 후 순간 부스터 발동을 성공하면 <br/>
일정 확률로 지속 시간이 긴 사이렌이 발동됩니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539">2024/5/30 패치</Link> 이후로 밸런스가 조정되었습니다.</Bq>

사이렌 효과가 5초나 지속되기 때문에, 발동만 성공한다면 굉장히 좋은 스킬입니다. 다만, 타이밍에 맞게 순간 부스터를 누르기가 조금 까다롭습니다. <br/>

24년 5월 패치로 발동 확률이 70%로 상향되었고, 발동 조건도 미사일 뿐만 아니라 물감옥, 넉백 등 착지 시 순간 부스터를 발동할 수 있는 모든 상황으로 확대되어 <br/>
스킬을 좀 더 사용할 수 있는 기회가 많아졌습니다. <br/> <br/> 

네오의 스킬에 피격당한 후에도 순간 부스터를 발동시킬 수 있기 때문에, 오리온 외의 카운터로 활용할 수도 있습니다.<br/> 
대신, 풍선이 장착된 상태에서 미사일 피격 시에는 발동 시킬 수 없습니다.

<br/>

단점으론 결국 공격 아이템을 맞아야 발동된다는 조건 때문에 순위가 애매하거나 아예 뒤쳐져버리면 할 수 있는게 없고, 바나나와 같이 미끄러지는 공격에는 <br/>
사이렌을 발동 시킬 수 없기 때문에 미끄러짐 공격을 하는 비비나 루시드 등의 캐릭터가 렉스의 카운터라고 할 수 있습니다.

<br/>

추가로, 팀전에서는 팀킬의 위험성이 있어 사용에 주의해야 합니다.
