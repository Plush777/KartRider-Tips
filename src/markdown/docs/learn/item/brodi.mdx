<Bq>브로디가 순식간에 부스터존을 뚝딱 만들어냅니다.<br/>
부스터존은 브로디가 바라보는 방향으로 설치되며 모두가 이용 가능합니다. <br/>
설치된 부스터존은 잠시 후 사라지며 밟으면 일시적으로 속도가 빨라집니다.</Bq>

부스터존 특성 상 카트바디를 강제로 밀어내는 식으로 동작하기 때문에, 가속도를 붙여 따라가는 용도 외에도 <br/>
예측 불가능한 곳에 설치하여 상대 레이서를 벽으로 몰아넣거나 낭떠러지로 추락시키는 등의 전략적인 활용이 가능합니다.<br/>

<br/>

대신 미사일, 사이렌 등에 피격 중이거나 부스터존 설치 지점이 공중인 경우 스킬이 취소 처리되어 그대로 사라지기 때문에 주의가 필요하며, <br/>
설치한 부스터 존은 팀 상관없이 모두가 영향을 받기 때문에 의도치 않은 팀킬 (주행 방해) 또는 상대방을 도와주는 상황 등 역효과가 발생할 수 있습니다.
