<Sprite className="character" backgroundPosition="-323px -1060px">데릭</Sprite>

<Bq>데릭이 극한에 치닫는 질주를 선보이며 자신이 가진 ‘전설의 레이서’ 타이틀을 증명해 보입니다.<br/>
일정 속도 이상으로 달릴 때 기본 속도가 LV 별로 상승합니다. <br/>
LV0 일 때는 기본 속도가 하락하지만,<br/><br/>
최대 LV3까지 LV이 상승할 때마다 기본 속도가 상승합니다.<br/>
주행에 방해를 받거나 아이템 슬롯이 무거워질 경우 LV이 초기화됩니다. <br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2574039">2024/5/30 패치</Link> 이후로 LV0일 때 기본 속도가 감소되었습니다.</Bq>

공격에 피격되지 않고 벽에 충돌하지 않으며 아이템을 1개 이하만 보유한 상태로 100km/h 이상을 유지하면 레벨이 오르며 최대 속도가 증가합니다.

레벨이 높아질수록 캐릭터와 카트바디 외형이 점점 붉어집니다. <br/>

최고속도 증가량이 높아서 레벨3에서는 속도가 300km/h에 도달해 부스터 이상의 속도를 유지하게 됩니다.

<br/>

대신, 레벨 초기화 조건들이 모두 아이템전에서 빈번하게 일어나는 상황이고, 트랙 난이도가 올라갈수록 높은 레벨을 유지해도 <br/>
벽에 안박고 주행하기가 쉽지 않기 때문에 운영 난이도가 높은 편입니다.
또한 레벨 유지를 위해서 아이템을 1개 이하로 보유해야 하기 때문에, <br/> 공격에 있어서 불리한 상황이 많습니다.

<br/>

불리한 상황도 많지만 속도에 대한 장점은 분명하여 공격을 받아 뒤로 처지더라도 레벨 회복이 빨라 <br/>
순위 회복이 빠른 편이며, 팀전의 경우 데릭을 러너로 보내주고 팀원들이 천사나 전자파로 보호하는 전략으로 활용하면 <br/>
절대 잡지 못하는 경우도 있습니다.

<br/>

트랙 폭이 넓은 맵의 경우, 실드 아이템 하나만 들고 주행할 때 맵 가장자리 (사이드)로 가서 아이템을 먹지 않고 <br/>
레벨을 유지하는 전략이 가능합니다.
