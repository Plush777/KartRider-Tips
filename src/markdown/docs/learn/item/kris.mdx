<Bq>크리스가 모든 상대 레이서의 아이템 슬롯을 해킹해 <br/>
아이템 하나를 바나나로 변경하는 장난을 칩니다.<br/>
해킹한 레이서의 수에 따라 아이템을 획득합니다.<br/><br/>
<Link target="_blank" href="https://kartdrift.nexon.com/kartdrift/ko/news/update/view?threadId=2527539">2024/3/21 노르테유 패치</Link> 이후로 크리스의 스킬 능력이 조정되었습니다.</Bq>

스킬 사용 시 상대 레이서 머리 위에 크리스 이모티콘이 뜨고, 3초 뒤 웃는 이모티콘으로 바뀌면서 지정 된 아이템 하나가 바나나로 변경됩니다.

(캐릭터 스킬로 획득한 특수 아이템도 포함)

<br/>

스킬을 사용하면 모든 상대 레이서의 아이템 하나를 무조건 날릴 수 있기 때문에 공격 아이템과의 궁합이 좋으며, 마지막 역전을 위한 자부 전략도 방해할 수 있습니다.

<br/>

이러한 점 때문에 개인전보단 팀전에서 주로 쓰이는 편입니다.
